# Airbnb-Pricing-and-Demand-Insights-New-York-2024
 
**Overview**:

This project analyzes 20,765 Airbnb listings in New York to find trends in pricing, room types, availability, and host behavior. The insights help both guests and hosts make better decisions.

**Objective**:

Understand room types, prices, and availability across neighborhoods
Study host behavior and listing patterns
Detect price outliers
Provide practical recommendations for guests and hosts

**Dataset**

Includes 22 features like id, name, host_name, neighborhood_group, price, room_type, reviews_per_month, and availability_365.

**Workflow**:

Data Cleaning: Handled missing values, corrected data types, and capped extreme prices.
EDA: Examined room types, price trends, neighborhood differences, availability patterns, and host activity.
Visualizations: Used bar charts, histograms, boxplots, heatmaps, and pairplots to spot trends and outliers.

**Key Insights**:

Entire homes/apartments are the most common and highest priced.
Manhattan is the most expensive borough; Brooklyn offers cheaper options.
Listings with high availability get more reviews.
Some hosts manage multiple listings, showing professional hosting trends.

**Recommendations**:

Guests: Choose listings with good availability and reviews.
Hosts: Optimize pricing, manage availability, and respond to reviews to attract bookings.

**Future Work**

Predict prices with machine learning
Analyze reviews to understand guest satisfaction
Build interactive dashboards for live monitoring
